# USDA Review Viewer

Static WebGL viewer for USDA files generated by `desmos2usd`.

## Serve

From the repo root:

```bash
python3 -m http.server <port>
```

Open:

```text
http://localhost:<port>/viewer/
```

For tailnet review, bind to all interfaces:

```bash
python3 -m http.server <port> --bind 0.0.0.0
```

Then open `http://<tailnet-host>:<port>/viewer/` from the phone.

## Use

- Pick an `artifacts/acceptance_local` sample from the dropdown.
- Or load a generated `.usda` with the file input or drag-drop.
- Orbit with mouse drag or one finger.
- Pan with right drag, Shift-drag, or two fingers.
- Zoom with mouse wheel or pinch.
- Click a prim or choose one in the list to inspect Desmos metadata.
- Large neutral underbounded panels are dimmed by default and excluded from the initial camera fit. Use `Panels` to cycle between dim, hide, and full rendering.
- Use `Focus` or `Hide` in the selection panel to inspect one prim or remove a distracting prim from the review view.
